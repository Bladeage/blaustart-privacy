# Blaustart — Privacy Policy

Hosts the privacy policy for the **Blaustart** Android app
(`com.farbigdev.blaustart`), served via GitHub Pages.

**Live:** https://bladeage.github.io/blaustart-privacy/

`index.html` is the bilingual (EN + DE) policy. The canonical source is
`docs/privacy.html` in the app repo — keep them in sync when the policy changes.
